This section refers to a premium feature.

Any Alien4Cloud properties can be set as a get_secret function property. Click on button to set a property as secret. The value of the secret should be the path to the desired value on the configurate Vault. Alien4Cloud 2.0.0-SM4 only support Vault as secret manager and only 1 Vault per location.
