We developed a cloudify plugin embedded with the alien4Cloud-cloudify3-provider, for scaling a LINUX compute type with Cloudify 3.3rc1.
Find here informations about how to set it up.

Cloudify 3.3rc1 manager

On the bootstraped manager, you have intall gcc and python-devel to run the following commands:

  • sudo yum install gcc
  • sudo yum install python-devel

Imports configuration

Make sure you have your imports on the orchestrator configuration set with theses values:

  • http://www.getcloudify.org/spec/cloudify/3.2/types.yaml
  • openstack-plugin.yaml
  • http://www.getcloudify.org/spec/diamond-plugin/1.3rc1/plugin.yaml

Now you are ready to scale a Linux type Compute. But there are some known limitations:

Known limitations

  • Volumes’ re-usage doesn’t work well. Will be fixed on the next release version.
